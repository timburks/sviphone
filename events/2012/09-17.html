It's about Data

Details

Our September meetup will be at Box in Los Altos, and our main topic is data and synchronization. How can we present rich data in our apps from network services, especially when our network connections are less than perfect.

Speakers:

Jens Alfke of Couchbase will present TouchDB (http://touchdb.org), an iOS port of the popular CouchDB database, which offers powerful synchronization features and the flexibility of a document database.

Michael Johnston of Simperium (https://simperium.com) will talk about Simperium's SDK for data synchronization.

Tim Anglade of Apigee will talk about Usergrid (http://apigee.com/about/developers), Apigee's platform for building APIs that you can run on their servers or host yourself.

And the Box API team will show us how to use Box One Cloud Libraries to connect applications to the cloud.
